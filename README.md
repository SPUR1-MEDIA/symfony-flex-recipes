# Spur1 Symfony Flex Recipes

This is a collection of recipes for Symfony Flex.

## Installation

```json
{
    "extra": {
        "symfony": {
            "allow-contrib": true,
            "endpoint": [
                "https://github.com/spur1/symfony-flex-recipes",
                "flex://defaults"
            ]
        }
    }
}

```
